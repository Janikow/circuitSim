⚡ Circuit Lab ready. Build a circuit and press RUN.

HOW TO USE

🔧
Place Components
Drag from palette onto the breadboard holes, or click a component then click a hole.
✏️
Wire Tool
Select Wire tool. Click a hole to start a wire, click another hole to finish it. Wires connect holes along rows.
▶
Run Simulation
Press RUN to simulate. LEDs glow, motors spin, buzzers buzz when current flows through them.
👆
Select & Move
Use Select tool to click and drag placed components to reposition them.
🗑️
Delete
Use Delete tool and click any component or wire to remove it.
🔘
Switch
Click a placed switch during simulation to toggle it on/off.
💡 Try: VCC → Resistor → LED → GND for a basic lit LED circuit!
